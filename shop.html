COSMIIC Power Module Circuit Board

Product Details

The COSMIIC Power Module is the central component of the Networked Neuroprosthetic (NNP) system. It serves as both the power source and communication hub for the entire implanted system, distributing power to all remote modules through the network cable.

Key Functions:

  • Houses rechargeable Li-ion batteries (3 cells in parallel)
  • Provides inductive charging through transcutaneous link
  • Contains MedRadio wireless link for communication and programming
  • Distributes power to all remote modules via network cable
  • 32-bit ARM-7 microprocessor with RTOS for signal processing
  • Magnetically activated emergency shutdown switch
  • Titanium case with hermetic feedthroughs
  • Two network ports for system expansion

Technical Specifications:

  • Processor: NXP Semiconductor LPC2129 (32-bit ARM-7)
  • Implant location: Torso (chest or abdomen)
  • Biocompatible titanium enclosure
  • Epoxy header for connections and antenna
  • Internal coil for inductive recharging
  • Polymer battery nest for component protection

Designed for chronic implantation in neuromodulation research, spinal cord injury rehabilitation, and early feasibility human studies. The power module's processing capabilities also support signal processing and data storage for remote module functions.

COSMIIC Remote Modules

Product Details

COSMIIC Remote Modules are networked implantable devices that can perform either electrical stimulation or neural recording functions. These modules are placed near target muscles, nerves, or brain regions and connect to the Power Module through the network cable system.

Module Capabilities:

  • Configurable for stimulation or recording applications
  • 8-channel linear connector design (stackable for higher counts)
  • Networked architecture for multi-module systems
  • Placed near target sites in arms, legs, or torso
  • Powered through the network cable from Power Module
  • Modular design allows system customization
  • Compatible with standard electrode arrays
  • Titanium biocompatible enclosure

Research Applications:

  • Spinal cord injury rehabilitation (hand grasp restoration)
  • Peripheral nerve stimulation and recording
  • Muscle activation for functional movement
  • Closed-loop neuromodulation systems
  • Brain-computer interface development
  • Autonomic nervous system regulation

Next Generation Modules:

COSMIIC is developing advanced remote modules including the HIVE (16-channel high-density module) and SWARM (miniaturized sensor modules for hands, feet, and organs) to expand research capabilities and enable new applications in bioelectronics.

COSMIIC Benchtop Development Kit

Product Details

The COSMIIC Benchtop Development Kit provides researchers with a complete platform for developing, testing, and validating neural interface systems before moving to in-vivo studies. This kit includes all necessary hardware, software, and documentation to get started quickly.

Kit Contents:

  • COSMIIC interface board with USB connectivity
  • Signal generator and load simulator modules
  • Test electrode arrays and connectors
  • Benchtop charging station
  • Software development kit (SDK) with APIs
  • Complete technical documentation and schematics
  • Example code and tutorials
  • Access to developer community forum

Capabilities:

  • Prototype custom stimulation and recording protocols
  • Test electrode impedance and channel integrity
  • Develop custom firmware modifications
  • Validate signal processing algorithms
  • Train lab personnel on system operation
  • Reduce in-vivo animal study requirements

Wireless Surface Stimulator

Product Details

Our Wireless Surface Stimulator is a non-invasive wearable device designed for transcutaneous electrical nerve stimulation (TENS) research and therapeutic applications. With programmable parameters and wireless control, this device is ideal for pain management studies, rehabilitation research, and human subject trials.

Key Features:

  • 4 independent stimulation channels
  • Wireless programming via smartphone or tablet
  • Rechargeable battery with 12+ hour operation
  • Comfortable, lightweight wearable design
  • Pre-programmed therapy protocols included
  • Data logging for compliance tracking
  • Compatible with standard surface electrodes
  • CE and FDA-registered device

Applications:

  • Chronic pain management research
  • Post-surgical rehabilitation studies
  • Functional electrical stimulation (FES)
  • Sports medicine and athletic recovery
  • Neuromuscular re-education
  • Human factors and usability testing
